What is difference between flex and grid ??

Flex: Flex is one dimensional. It's work by default only row. By changing flex-direction we can change its direction to column or reverse way.

Grid: Grid is a two dimensional layout. It work rows and column at the same time.

What is difference between Bootstrap and Tailwind Css ??

Bootstrap and Tailwind CSS are Css frameworks.But there are some difference between them.

Differences are:

Points Bootstrap Tailwind Css
01. customization Bootstrap is not more customizable then tailwind Tailwind is customizable
02. utilities Bootstrap have utilities but not more then tailwind css. tailwind css have largest utilities
03. Compenent library Bootstrap have some readymate component library. It very useful for user and its free . Tailwind also have component library but they provides primium services.
